* Version 1.0.0

Script de practica 
***


## procedimiento
```
1- Al iniciar la conversacion con el comando /start saluda a quien lo contacta capturando el primer nombre de la persona
por medio del codigo {update.effective_user.first_name}

2- Con el comando /frase nos devuelve una frase random que toma de un array de frases

3- Con el comando /cancion nos despliega 10 opciones por medio de botones y al seleccionar alguno nos devuelve un texto 
recomendando una cancion o disco

4- Con el comando /qr nos pide que ingresemos un texto para convertirlo en codigo Qr. Toma el texto ingresado, genera el 
qr, lo envia en el chat y borra el archivo de la carpeta

```
